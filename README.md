# 소비재 기업 감사보수 결정요인 분석

소비재(식품·음료·생활용품·화장품) 분야 상장기업의 재무데이터와 감사보수 데이터를 통합·분석하여,
어떤 요인이 감사보수(Audit Fee)에 영향을 미치는지 다양한 머신러닝 회귀 모델로 예측·분석한
감사(Audit) 분야 취업 포트폴리오입니다.

## 배경

감사보수는 기업 규모, 감사 위험, 감사인 유형(Big4 여부) 등 다양한 요인에 의해 결정된다는 것이
회계·감사 분야의 잘 알려진 연구 주제입니다. 이 프로젝트는 DART(전자공시시스템) 공개 데이터를
활용해 이 결정요인을 데이터로 직접 확인해보는 것을 목표로 합니다.

## ⚠️ 데이터에 대한 중요 안내

- 이 저장소에 포함된 CSV(`소비재_재무정보.csv`, `소비재_감사보수정보.csv`)는 **실제 DART 데이터가
  아닌 가상으로 생성한 샘플 데이터**입니다. (자산 규모와 Big4 여부가 감사보수에 영향을 주도록
  설계했지만, 실제 수치·비율과는 다릅니다.)
- 실제 데이터로 분석하려면 `dart_audit_fee_collector.py`를 본인의 DART API 키로 실행하세요.
  단, 감사보수 API(`adtServcCnclsSttus`)의 정확한 응답 컬럼명은 이 프로젝트 작성 시점에
  실시간 검증하지 못했으므로, 스크립트 실행 후 `audit_raw_sample.csv`로 저장되는 원본 응답을
  먼저 확인하고 스크립트 상단의 `COLUMN MAPPING` 부분을 실제 컬럼명에 맞게 수정해야 합니다.

## 분석 대상 기업 (12개사)

CJ제일제당, 오뚜기, 농심, 오리온, 하이트진로, 롯데칠성음료, LG생활건강, 애경산업,
아모레퍼시픽, 한국콜마, 매일유업, 빙그레 (2018~2023년, 6개년)

## 분석 절차

1. 기본 라이브러리 준비
2. 데이터 수집 및 통합 (재무정보 + 감사보수정보 병합)
3. 범주형 변수 EDA (감사인·감사의견·업종 분포)
4. 범주형 변수 이진화 (Big4 여부, 감사의견 적정 여부)
5. 상관관계 분석 → 회계항등식(자산=부채+자본)으로 인한 다중공선성 발견
6. 불필요 변수 제거 (equity 제거, assets→log_assets, debt→leverage 변환)
7. 결측치 처리
8. 범주형 변수 인코딩 (get_dummies)
9. 데이터 분할 및 정규화 (train_test_split + StandardScaler)
10. 머신러닝 회귀 모델 학습 (LinearRegression, RandomForest, GradientBoosting, XGBoost)
11. 모델 성능 평가 및 최적 모델 선정 (MAE, RMSE, R²)
12. 딥러닝(Keras) 모델 학습 곡선 시각화를 통한 학습 상태 점검

## 주요 결과 요약

- **Big4 여부**가 감사보수에 가장 큰 영향을 미치는 요인으로 나타났습니다 (상관계수 0.83, RandomForest
  feature importance 약 69%).
- **자산 규모(log_assets)**도 유의미한 양(+)의 영향을 보였으나(상관계수 0.52), Big4 여부에 비해
  상대적으로 작은 비중(feature importance 약 12%)을 차지했습니다.
- 분석 과정에서 `assets`·`debt`·`equity`가 회계항등식으로 인해 서로 강하게 상관되어 있음을
  발견하고, 이를 `log_assets`(로그 규모)와 `leverage`(부채비율)로 재정의해 다중공선성 문제를
  해소했습니다.

자세한 분석 과정과 해석은 [`REPORT.md`](./REPORT.md)를 참고하세요.

## 한계

- 표본 수가 12개사 x 6개년(72건)으로 적어 통계적 일반화에는 한계가 있습니다.
- 감사보수 데이터가 실제 DART 응답이 아닌 가상 샘플 데이터이므로, Big4 프리미엄과 기업 규모
  효과의 상대적 비율은 실제 데이터로 교체 시 달라질 수 있습니다.
- 본 프로젝트는 감사보수 결정요인을 데이터 기반으로 탐색하는 **분석 접근법과 코딩 역량**을
  보여주는 데 목적이 있으며, 실제 감사보수 산정이나 감사계획 수립에 사용하기 위한 것이 아닙니다.
